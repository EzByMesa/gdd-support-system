# 06. Руководство пользователя

Это руководство предназначено для пользователей с ролью **USER** — клиентов службы поддержки.

---

## Регистрация и вход

### Регистрация

1. Откройте систему в браузере (например, `http://localhost`)
2. Нажмите ссылку **"Зарегистрироваться"** на странице входа
3. Заполните форму:
   - **Логин** — латиница, цифры, символы `_`, `.`, `-`, минимум 3 символа
   - **Пароль** — минимум 8 символов
   - **Отображаемое имя** — как вас будут видеть в системе
   - **Email** — необязательно
4. Нажмите **"Зарегистрироваться"**

После успешной регистрации вы автоматически входите в систему с ролью **USER**.

> Если кнопка регистрации не отображается, значит самостоятельная регистрация отключена администратором. Обратитесь к администратору для создания учётной записи.

### Вход

1. Введите логин и пароль
2. Нажмите **"Войти"**

Сессия сохраняется в браузере на 30 дней. При следующем открытии сайта вход происходит автоматически.

### Вход через 1С

Если в системе настроена интеграция с 1С, на странице входа появится дополнительная кнопка. Введите логин и пароль от вашей учётной записи 1С.

### Выход

Нажмите кнопку **"Выйти"** в боковом меню или в шапке страницы.

---

## Создание обращения

1. Нажмите кнопку **"+ Новое обращение"** на главной странице
2. Заполните форму:
   - **Тема** — краткое описание проблемы (обязательно)
   - **Описание** — подробное изложение проблемы (обязательно)
   - **Приоритет** — Низкий / Средний (по умолчанию) / Высокий / Критический
3. При необходимости прикрепите файлы
4. Нажмите **"Создать обращение"**

После создания обращение получает уникальный номер (например, `#42`) и отображается в вашем списке со статусом **"Открытое"**.

### Прикрепление файлов

Вы можете прикрепить файлы при создании обращения:
- Максимальный размер одного файла: **50 МБ**
- Файлы хранятся в зашифрованном виде на сервере

## Просмотр обращений

### Список обращений

Главная страница (`/`) показывает **все ваши обращения**. Для каждого обращения отображается:
- Номер (`#42`)
- Заголовок
- Статус (цветной бейдж)
- Приоритет
- Тематическая группа (если определена системой)
- Дата создания

### Фильтры по статусу

Над списком расположены вкладки для фильтрации:

| Вкладка | Что показывает |
|---------|---------------|
| Все | Все ваши обращения |
| Открытые | `OPEN` — новые, агент ещё не назначен |
| В работе | `IN_PROGRESS` — агент работает над проблемой |
| Ожидают ответа | `WAITING_FOR_USER` — агент задал вопрос, ждёт вашего ответа |
| Решённые | `RESOLVED` — проблема решена |
| Закрытые | `CLOSED` — обращение закрыто |

### Пагинация

При большом количестве обращений в нижней части списка появляются кнопки навигации по страницам (по 20 обращений на странице).

---

## Просмотр обращения и чат с агентом

Нажмите на обращение в списке, чтобы открыть его детальную страницу.

### Информационная панель

В правой части страницы показаны:
- Текущий статус
- Приоритет
- Дата создания
- Агент (отображается псевдоним, например "Ласковый Котёнок")

> Из соображений конфиденциальности вы видите только псевдоним агента, но не его реальное имя.

### Описание обращения

Ваш исходный текст и прикреплённые файлы отображаются в основной области страницы.

### Скачивание вложений

Нажмите на имя файла в описании или в сообщении чата, чтобы скачать его.

### Чат с агентом

В нижней части страницы расположено поле чата:

1. Введите сообщение в текстовое поле
2. При необходимости прикрепите файлы (кнопка скрепки)
3. Нажмите **"Отправить"** или клавишу **Enter**

**Особенности чата:**
- Сообщения отображаются в режиме реального времени (WebSocket)
- Если агент печатает ответ, появляется индикатор набора ("Ласковый Котёнок печатает...")
- Вы увидите все сообщения истории при открытии страницы (последние 50)

### Автоматическая смена статуса

Если агент перевёл обращение в статус **"Ожидает ответа"** и вы написали сообщение в чат, статус автоматически меняется обратно на **"В работе"**. Агент увидит, что вы ответили.

## Уведомления

### Колокольчик

В правом верхнем углу страницы расположен значок колокольчика. Рядом с ним отображается количество непрочитанных уведомлений.

Нажмите на колокольчик, чтобы открыть список уведомлений.

### Типы уведомлений

| Уведомление | Когда приходит |
|-------------|---------------|
| Новое сообщение | Агент ответил в чате |
| Статус изменён | Агент изменил статус вашего обращения |
| Агент изменён | После переназначения (делегирования) тикета |

### Web Push уведомления

Система может отправлять уведомления даже когда браузер закрыт (если ваш браузер поддерживает Web Push):

1. Система автоматически предложит разрешить уведомления при первом входе
2. Нажмите **"Разрешить"** в диалоге браузера
3. Уведомления будут приходить как системные оповещения браузера

Для отключения Web Push откажитесь от разрешения или отзовите его в настройках браузера.

### Отметить прочитанным

- Нажмите на отдельное уведомление, чтобы отметить его прочитанным
- Нажмите **"Отметить все прочитанными"** для массовой отметки

## Часто задаваемые вопросы

**Почему я не вижу реальное имя агента?**
Это сделано намеренно для защиты данных сотрудников. Вы общаетесь с псевдонимом ("Ласковый Котёнок"), который уникален для вашего обращения.

**Почему псевдоним агента изменился?**
Ваше обращение было передано другому агенту (делегировано). Новый агент получает новый псевдоним.

**Могу ли я изменить статус обращения?**
Нет. Статус управляется агентами. Вы можете косвенно влиять на статус: ответ в чат при статусе "Ожидает ответа" автоматически меняет статус на "В работе".

**Я не вижу кнопку регистрации. Что делать?**
Самостоятельная регистрация отключена. Обратитесь к администратору вашей организации для создания учётной записи.
